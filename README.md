# CSS Exercises

This project is part of **The Odin Project** curriculum and contains my solutions to the CSS exercises from the Foundations, Intermediate HTML & CSS, and Advanced HTML & CSS sections.

By completing these exercises, I demonstrated my understanding of core CSS concepts, including selectors, the cascade, the box model, typography, layout techniques, Flexbox, Grid, responsive design, and modern CSS practices. I also strengthened my ability to structure stylesheets, analyze requirements, and translate visual designs into functional user interfaces.

This repository serves as a record of my progress and growth in front-end development throughout the course.
