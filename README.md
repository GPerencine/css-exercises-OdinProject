# CSS Exercises

This project is part of **The Odin Project** curriculum and contains my solutions to the CSS exercises from the Foundations, Intermediate HTML & CSS, and Advanced HTML & CSS sections.

By completing these exercises, I demonstrated my understanding of core CSS concepts, including selectors, the cascade, the box model, typography, layout techniques, Flexbox, Grid, responsive design, and modern CSS practices. I also strengthened my ability to structure stylesheets, analyze requirements, and translate visual designs into functional user interfaces.

This repository serves as a record of my progress and growth in front-end development throughout the course.

## Repository Structure

The repository is divided into three main sections following the progression of The Odin Project curriculum:

* `foundations/`  contains exercises covering fundamental CSS concepts and introductory layout techniques.
* `intermediate-html-css/` contains exercises focused on more advanced styling concepts, responsive design, and modern layout approaches.
* `advanced-html-css/` contains exercises exploring advanced CSS topics and professional front-end development practices.

Within each section:

* The second level represents the topic (for example, `intro-to-css`).
* The third level contains the individual exercises for that topic (for example, `01-css-methods`).
* Inside each exercise folder are the original `README.md` containing the instructions and requirements, along with my solution to the exercise.

Example:

```text
foundations/
└── intro-to-css/
    └── 01-css-methods/
        ├── index.html
        ├── README.md
        └── style.css
```

This organization keeps the repository aligned with the course structure and makes it easy to track my progress throughout the curriculum.
